# react-stateless

## About

This project is single-page react app for my [stateless express backend server](https://github.com/ensarkr/express-stateless). These both projects are created to deepen my understanding of stateless backend. React app has very simple ui with little to no styling.

## Features

If you wanna read features of backend server you can check readme file of [express-stateless](https://github.com/ensarkr/express-stateless).

## To Run Locally

```
$ npm install
$ npm run dev
```

After this react site should be accessible on http://localhost:5173/.
You also need to download and serve my backend server. Here is a [link](https://github.com/ensarkr/express-stateless).

- React
